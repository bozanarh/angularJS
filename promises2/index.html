promises in angularJS

To create your own promises, you need to use $q service.

  • q.defer() returns defered object
  • call resolve() on defered object to resolve the promise
  • call reject() on defered object to reject the promise
  • return promise
    • your promise handler has to call then() function on the promise and there you have to implement success and failure method
  • you can link multiple promises, but each of them has to return promise

Note, promise runs only once, but here every time, you click on the button, you create it.
