Passing arguments from the directive to the controller function

This example shows how to invoke controller function from the directive and how to pass parameters from directive to the controller

Pay attention to the following:

  • In index.html you have "ctrlHanlder", NOT ctrlHandler()
  • In directive you have defined controller because you need to do some processing
  • In the directive you are calling outer controller function(ctrlHandler) by invoking $scope.add()(argument)
